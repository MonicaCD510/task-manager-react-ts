# Task Manager React TypeScript App

## Overview
This project is a Task Manager application built with React, TypeScript, and Vite. The application demonstrates list rendering, conditional rendering, filtering, component composition, and TypeScript interfaces.

## Features
- Display a list of tasks
- Toggle task status
- Delete tasks
- Filter tasks by status
- Component-based structure
- TypeScript type safety

## Components
### TaskList
Displays the list of tasks.

### TaskItem
Displays an individual task with status and delete controls.

### TaskFilter
Allows filtering tasks by status.

## Technologies Used
- React
- TypeScript
- Vite
- CSS
